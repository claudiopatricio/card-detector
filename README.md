# Simple card detector using OpenCV

![CARD-DETECTOR](https://github.com/claudiopatricio/card-detector/blob/master/sample.jpg)

This code was developed in CodeBlocks using only OpenCV (no advanced libraries).

You can watch [this video](https://youtu.be/WW-dpt87Dyo) as a sample of working, it was only design to work with our deck, but should work with similar decks.

Co-Author: Pedro Abade

## Required software

* [CodeBlocks](http://www.codeblocks.org/)
* [OpenCV](https://opencv.org/) (You can also install the last version from [here](https://github.com/jayrambhia/Install-OpenCV))
